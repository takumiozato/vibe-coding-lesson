# IRメディアサイト - ディレクトリ構成

## ルートディレクトリ構成
```
/ir-media-site
  ├── /src
  │    ├── /pages
  │    │    ├── index.astro            # 記事一覧ページ
  │    │    └── /articles/[slug].astro # 記事詳細ページ
  │    ├── /content
  │    │    └── /articles
  │    │         ├── article1.md  # 記事1のMarkdownファイル
  │    │         ├── article2.md  # 記事2のMarkdownファイル
  │    │         └── ...          # その他の記事ファイル
  │    ├── /styles
  │    │    └── global.css         # グローバルCSS
  │    └── /components
  │         └── ArticleCard.astro  # 記事カードコンポーネント
  ├── /public
  │    └── /images                 # 画像等の静的ファイル
  ├── package.json                 # プロジェクト設定
  ├── tailwind.config.js           # Tailwind設定
  └── astro.config.mjs             # Astro設定
```
